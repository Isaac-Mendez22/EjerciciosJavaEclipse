```
Temas vistos en los Ejercicios:
1. Crear el primer projecto y clase en java
2. convercion y tipos de datos (implicito y explicito)
3. Operadores ([], --, ++, +=, -=, &&)
4. Switch Case
5. ciclos (For, While, Do While, Break, Continue)
6. Metodos(metodos  y sobrecarga de metodos)
7. Ejercicio CalculatorBrain
8. Colecciones (Arrays, List-ArrayList, Set-HashSet, Map-HashMap, Iterator)
9. Clases y Objetos (constructor, overloading contructor,propiedades y metodos)
10. Paquetes (modificadores de acceso) 
11. Encapsulamiento (get, set)
12. Excepciones/Errores (sintaxis, runtime, logica) = (POOProyect, ExceptionProyect,)
```
